CSS Text Formating

Text Color

Text color can be changed on any text element like <h1>, <p>

The color property is used to set the color of the text. The color is specified by:

  • Name (i.e.: green)
  • Hex code (#FFC0CB)
  • Decimal code (rgb(255, 165, 0))
  • Hsl (hsl(270, 50%, 60%))

Example:

h1 {

color:

green;

}

Text Alignment

The text-align property is used to set the horizontal alignment of a text.p>

A text can be left or right aligned, centered, or justified.color is specified by:

The alignment is specified by:

  • left
  • center
  • right

Example:

h1 {

text-align:

center;

}

Text Decoration

The text-decoration property is used to set or remove decorations from text.

The value text-decoration:none; is often used to remove underlines from links:

These other text-decoration values are used to decorate text:

  • overline
  • line-through
  • underline

Example:

h1 {

text-align:

line-through;

}

Text Transformation

The text-transform property is used to specify uppercase and lowercase letters in a text.

It can be used to turn everything into uppercase or lowercase letters, or capitalize the first letter of each word

  • uppercase
  • lowercase
  • capitalize

Example:

h1 {

text-transform:

capitalize;

}

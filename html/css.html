CSS Introduction

CSS stands for Cascading Style Sheets

Used for styling our web page

Css general syntax:

selector {

property:

value;

}

There are three different ways to add styling to your document:

  • Inline CSS

    • Example:

    <h1 style = "

    color:

    red;

    "> WSU </h1>
  • Embedded CSS

    • Example:

    <style>
    h1 {

    color:

    purple;

    }
    </style>
  • External CSS

    • Example:

    <style href= "external_file.css" rel= "stylesheet" type = "text/css" >

The precedence order

Inline > Embedded > External

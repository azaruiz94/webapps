HTML Tags

<tagname>content goes here...</tagname>

HTML tags are element names surrounded by angle brackets:

  • HTML tags normally come in pairs like <p> and </p>
  • The first tag in a pair is the start tag, the second tag is the end tag
  • The end tag is written like the start tag, but with a forward slash inserted before the tag name

Note: The start tag is also called the opening tag, and the end tag the closing tag.

HTML Headings

Headings are defined with the <h1> to <h6> tags.

Usage:

<h2>Heading 2<h2>
<h3>Heading 3<h3>
<h4>Heading 4<h4>
<h5>Heading 5<h5>
<h6>Heading 6<h6>

Heading 1

Heading 3

HTML Paragraphs

The HTML <p> element defines a paragraph

A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after a paragraph.

Usage:

<p>This is a paragraph.</p>

<p>This is another paragraph.</p>

Example:

This is a paragraph.

This is another paragraph.

HTML Cite

The <cite> tag defines the title of a work (e.g. a book, a song, a movie, a painting, a sculpture, etc.).

Usage:

<p><cite>The Scream</cite> by Edward Munch. Painted in 1893.</p>

Example:

The Scream by Edward Munch. Painted in 1893.

HTML Lists

Unordered List

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag.

The list items will be marked with bullets (small black circles) by default:

Usage:

<ul>

<li>Coffee</li>

<li>Tea</li>

<li>Milk</li>

</ul>


Example:

  • Coffee
  • Tea
  • Milk
Ordered List

An ordered list starts with the <ol> tag. Each list item starts with the <li> tag.

The list items will be marked with numbers by default.

Usage:

<ol>

<li>Coffee</li>

<li>Tea</li>

<li>Milk</li>

</ol>


Example:

  1. Coffee
  2. Tea
  3. Milk
Ordered HTML List - The Type Attribute

The type attribute of the <ol> tag, defines the type of the list item marker:

Type Desciption
type="1" The list items will be numbered with numbers (default)
type="A" The list items will be numbered with uppercase letters
type="a" The list items will be numbered with lowercase letters
type="I" The list items will be numbered with uppercase roman numbers
type="i" The list items will be numbered with lowercase roman numbers

HTML Table

HTML tables allow web authors to arrange data into rows and columns.

An HTML table is defined with the <table> tag.

Each table row is defined with the <tr> tag. A table header is defined with the <th>tag. By default, table headings are bold and centered. A table data/cell is defined with the <td> tag.

Usage:

<table class="table"> <tr> <th>Firstname</th> <th>Lastname</th> <th>Age</th> </tr> <tr> <td>Jill</td> <td>Smith</td> <td>50</td> </tr> <tr> <td>Eve</td> <td>Jackson</td> <td>94</td> </tr> </table>

Example:

Firstname Lastname Age
Jill Smith 50
Eve Jackson 94

HTML Links

HTML links are hyperlinks.

You can click on a link and jump to another document.

When you move the mouse over a link, the mouse arrow will turn into a little hand.

Link tag for absolute URL

Usage:

<a href="https://www.google.com/"> Go to Google </a>;

Example:

Go to Google
Link tag for relative URL

Usage:

<a href="Css.html"> Go to CSS site</a >;

Example:

Go to CSS site
Link tag for relative URL - Specific section on the same page

Usage:

<a href="#object_id"> Go to a specific section on this page</a >;

Example:

Go to Headings section

HTML Images

In HTML, images are defined with the <img> tag.

The <img> tag is empty, it contains attributes only, and does not have a closing tag.

The src attribute specifies the URL (web address) of the image:

The alt attribute provides an alternate text for an image, if the user for some reason cannot view it (because of slow connection, an error in the src attribute, or if the user uses a screen reader).

The value of the alt attribute should describe the image.

Usage:

CSS Selectors

CSS selectors are used to "find" (or select) the HTML elements you want to style.

Element selector (less customizable)

The element selector selects HTML elements based on the element name.

Example:

h1 {

color:

green;

}

ID selector

The id selector uses the id attribute of an HTML element to select a specific element.

The id of an element is unique within a page, so the id selector is used to select one unique element!

To select an element with a specific id, write a hash (#) character, followed by the id of the element.

Example:

<h1 id= "onlythis">Only this<h1>
#onlythis {

color:

green;

}

Class selector

The class selector selects HTML elements with a specific class attribute.

To select elements with a specific class, write a period (.) character, followed by the class name.

Example:

<h1 class= "onlythis">Only this<h1>
<h1 class= "onlythis">Also this<h1>
<h1>Not this this<h1>
#onlythis {

color:

green;

}

*Only the first two elements will be styled

Group selector (used to minimize code)

The grouping selector selects all the HTML elements with the same style definitions.

Example:

p.onlythis, h1.maybethis {

color:

orange;

text-align:

center;

}

Universal selector (used to minimize code)

The universal selector (*) selects all HTML elements on the page.

Example:

* {

color:

orange;

text-align:

center;

}
